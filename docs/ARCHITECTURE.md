
# ARCHITECTURE


## Visão geral

O projeto será dividido em módulos:


- app/core: configuração, constantes e logging

- app/api: rotas da API

- app/services: regras de negócio (fase posterior)

- app/automations: automação Playwright (fase posterior)

- docs: documentação do projeto


## Stack

- FastAPI para API

- PostgreSQL para persistência

- Redis para apoio operacional

- Playwright para automação web
